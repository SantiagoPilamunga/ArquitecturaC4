Para la ejecucion primero lo hacemos con el backend cambiamos de directorio hasta encontrarnos en consulta dentro de backend y ahi ejecutamos los comandos "mvn clean install",
despues de que se ejecute usamos "mvn spring-boot:run" luego abrimos otra terminal para iniciar el frontend cambiamos el directorio para estar en frontend,
una vez ahi ejecutamos el comando "npm install" si no lo tenemos, sino solamente ejecutamos "npm start" y ahi directamente en el terminal nos va a salir
donde se inicio damos clic y nos llevara a la pagina web correspondiente
